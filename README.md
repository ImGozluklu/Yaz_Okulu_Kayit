# Yaz Okulu Kayıt

## Projenin Amacı
- Öğrencilerin yaz okuluna kayıt durumlarının tutulması. 
- Ders açılma taleplerine bakılması.
- Online bir stok sağlanır.

## Kullanılan Teknolojiler , Kütüphaneler ve Yöntemler
- HTML
- CSS
- ASP.NET
- C#
- Bootstrap
